About Me

Huffman Compression/Decompression Suite

Java

In a project for my Data Structures and Algorithms course, I implemented a Huffman coding suite capable of compressing and decompressing files using Huffman trees and prefix codes.

In more recent years, I have taught other students how to code this assignment as a teaching assistant for the Data Structures and Algorithms course.

Trie Autocomplete Algorithm

Java

In this project for my Data Structures and Algorithms course, I wrote a program that uses binary search algorithms and trie data structures to implement an Autocomplete algorithm that guesses the user's query based on the prefix they have typed and a weight that corresponds to the popularity of the word.

In more recent years, I have taught other students how to code this assignment as a teaching assistant for the Data Structures and Algorithms course.

Simulation of Single-Level Cache

Java

In this project for my Computer Architecture course, I designed and implemented a program to simulate loads and stores to a single-level cache. The simulation can create caches with varying associativities and block sizes.

User Recommendation Algorithm

Python

In this project for my Introduction to Programming course, I created a program that recommends items to users based on analysis of data about their preferences. The program reads in data about user preferences and then, given a specific user, makes recommendations to them based on the preferences of other users who liked similar items.

In more recent years, I have taught other students how to code this assignment as a teaching assistant for the Introduction to Programming course.
