Cellular Automata Simulator

Java, Java FX, XML

I worked with a team of 2 others to design a program capable of simulating and displaying several different types of cellular automata.

Stats about the states of the cells in the simulation grid are displayed visually on a graph. The user can choose between a fire spreading simulation, a game of life simulation, a predator-prey simulation, a segregation simulation, and a slime mold simulation. The program gives the user the ability to set multiple variables about the situation being simulated, as the ability to change the simulation speed, to pause and then resume the simulation, and to move through the simulation step by step.

Simple Logo Program

Java, Java FX

I worked with a team of 3 others to design a turtle based coding environment that uses a simple version of the Logo programming language.

The program supports function creation, variable assignment, and loops with variable iteration patterns. It also allows the user to load preset turtle functions and also to save their own functions to file. It also allows the user to save and load their workspace so that the set of functions they have defined can be accessed again when they reload the program.

Game Authoring Environment

Java, Java FX, XML

I worked with a team of 9 others to design a game environment that uses an entity-component system to allow a user to create multi-level games in a game editor, store them to their local computer, and then load and play them.

Users can give their game characters attributes, such as health, set character abilities to keys or to video game controller inputs, and choose from multiple different types of win conditions for the game. They can also add music or interesting backgrounds to their games.It also gives the user the ability to reopen a game they've created in the editor and to save their progress in multiple games. The game play environment supports split screen for multiple players.
